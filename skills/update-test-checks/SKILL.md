# Update Test Checks

Only regenerate CHECK lines when the test declares that assertions are autogenerated. Do not use this skill to hide semantic regressions.

## Workflow

1. Inspect the first lines of the test for `Assertions have been autogenerated`.
2. Run the helper in dry mode to identify the correct LLVM update script.
3. If the failure is pure CHECK drift, rerun with `--apply`.
4. Review the diff. Escalate if output changes semantics or involves metax-only behavior.

## Command

```bash
python3 skills/update-test-checks/scripts/update_checks.py \
  llvm/test/CodeGen/Metax/foo.ll \
  --llvm-source /path/to/llvm-project \
  --llvm-build /path/to/build
```

Apply after review:

```bash
python3 skills/update-test-checks/scripts/update_checks.py TEST --llvm-source SRC --llvm-build BUILD --apply
```
